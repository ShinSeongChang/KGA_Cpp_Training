조별과제는 주제를 정해서 스터디 그룹끼리 자료조사 -> ppt 만들어서 -> 발표까지

주제는

[메모리]

힙 메모리란 무엇인지? / 왜 알아야 하는지? / 전역변수와 어떤 관계가 있는지? / 전역변수란 무엇인지?


[개별과제]

게임 루프 활용해서 배틀 시스템 + 걷기 시스템 만들기

플레이어의 최대 Hp는 50, 플레이어는 6번 길을 걸으면 게임 종료.

걷기에 대해서 -> 사용자는 랜덤하게 진행한다. (길 (Load), 강 (River), 산 (Mountain) )
	      -> 30% 확률로 길로 간다. -> 랜덤 값 리롤 (어디로 갈지)
	      -> 20% 확률로 강으로 간다. -> 랜덤 값 리롤 (어디로 갈지)
	      -> 50% 확률로 산으로 간다. -> 전투 발생 -> 랜덤 값 리롤 (어디로 갈지)

전투에 대해서 -> 몬스터가 등장한다. 전투는 Auto로 진행한다.
	      -> 몬스터의 체력, 공격력, 플레이어의 공격력은 Custom 한다.
	      -> 전투는 40% 확률로 발생하고
	      -> 60% 확률로 플레이어가 회복한다. 회복량은 Custom 한다.

Deadline은 월요일 수업 전까지
